# Dataset Methodology

This dataset supports an informal game, not a scientific AI detector benchmark.

Human snippets are intended to come from permissively licensed open-source repositories at pinned pre-2021 commits. AI snippets are generated from structured behavioral specs derived from normalized, anonymized human display code. Both sides are normalized with comments stripped and top-level names anonymized before review.

Generated code is accepted or rejected by human review only. Accepted AI snippets are not manually edited.
